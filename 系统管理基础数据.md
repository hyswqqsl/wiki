# 系统管理基础数据接口

## 一 行政区基础数据添加，RegionController
1. 取得zTree结构json，/region/admin/tree,GET
    * **web使用**
    * **角色：市河长办管理员**  
    * 得到前台显示用的region属性结构json串
    * 参数：
       * code,行政区code
    * 返回：
        * OK，json串
2. 新建节点，/region/admin/node/create,POST
    * **web使用**
    * **角色：市河长办管理员**  
    * 参数：
        * parentId,父级节点id
        * name,节点名
    * 返回：
        * OK,删除成功,返回新建的节点{id:xx,name:xxx,pId:xxx}
        * DATA_NOEXIST,parengId不存在              
3. 修改节点，/region/admin/node/update,POST
    * **web使用**
    * **角色：市河长办管理员**  
    * 参数：
        * id,节点id
        * name,节点名
        * code,编码
        * remark,备注
    * 返回：
        * OK,删除成功
        * DATA_NOEXIST,id不存在             
4. 删除节点，/region/admin/node/delete/{id},DELETE
    * **web使用**
    * **角色：市河长办管理员**  
    * 参数：
        * id,节点id
    * 返回:
        * OK,删除成功
        * DATA_NOEXIST,id不存在 
        
## 二 河长编辑接口，HzUserController        
1. 新建河长，/hzUser/admin/create,POST
   * **web端使用**
   * **角色：市河长办管理员**
   * 参数：
       * name
       * code，人员编码
       * phone，手机号
       * type，类型
       * remark，备注
       * regionId，行政区id
   * 返回：
       * OK，新建成功
2. 编辑河长，/hzUser/admin/update,POST
   * **web端使用**
   * **角色：市河长办管理员**
   * 参数：
       * id
       * name
       * code，人员编码
       * phone，手机号
       * type，类型
       * remark，备注
   * 返回：
       * OK，修改成功
       * DATA_NOEXIST           
3. 删除河长,/hzUser/admin/delete/{id},DELETE
   * **web端使用**
   * **角色：市河长办管理员**
   * 参数：
       * id
   * 返回：
       * OK，删除成功 
       * DATA_NOEXIST
4. 取得河长列表，/hzUser/admin/lists,GET       
   * **web端使用**
   * **角色：市河长办管理员**
   * 参数：
       * regionId,行政区id
   * 返回：
       * OK，删除成功 
       * DATA_NOEXIST

三 河长办编辑接口,HzbController
1. 新建河长办,/hzb/create,POST
   * **web端使用**
   * **角色：市河长办管理员**
   * 只有市，县，乡能建立河长办
   * 参数：
       * name 河长办名称
       * phone，河长办电话
       * remark，备注
       * regionId，行政区id
   * 返回：
       * OK，新建成功
       * FAIL，不是市县乡行政区，建立失败
2. 编辑河长办,/hzb/update,POST
   * **web端使用**
   * **角色：市河长办管理员**
   * 参数：
       * id
       * name
       * phone，手机号
       * remark，备注
   * 返回：
       * OK，修改成功
       * DATA_NOEXIST
3. 删除河长办,/hzb/delete,DELETE
   * **web端使用**
   * **角色：市河长办管理员**
   * 删除河长办时，同时删除河长办用户
   * 参数：
       * id
   * 返回：
       * OK，删除成功
       * DATA_NOEXIST
